发票批量下载工具 绿色版

使用方式：
1. 双击 InvoiceDownloader.exe 启动。
2. 在工具里设置“二维码目录”和“发票输出目录”。
3. 点击“开始下载”即可批量下载发票。

说明：
- 本绿色版无需安装，解压后即可运行。
- 请保持 InvoiceDownloader.exe 与 _internal 文件夹在同一目录下。
- 如果要发给别人，建议发送整个绿色版文件夹，或发送对应 zip 压缩包。

目录建议：
- 二维码目录：可把二维码图片放到“示例_二维码目录”里，或自行选择其他目录。
- 发票输出目录：可选择“示例_发票输出目录”，或自行选择其他目录。

注意：
- 不要单独移动或删除 _internal 文件夹，否则程序可能无法启动。
- 首次运行若被系统拦截，请选择“仍要运行”。
